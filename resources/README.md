# Resources Folder

## Images
Place your custom images in the `images/` folder to use them for communication buttons.

Supported formats: JPG, PNG, GIF, WebP

### Recommended Image Sizes
- Minimum: 200x200 pixels
- Recommended: 400x400 pixels
- Format: Square images work best

### Usage
1. Add your images to the `resources/images/` folder
2. Open the app settings (⚙️ icon)
3. Scroll to "Customize Button Images"
4. Select a button and choose your image from the file picker
